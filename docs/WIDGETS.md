# Teacher IDE Widgets

All widgets live in `packages/teacher-core/src/browser/widgets/` unless noted.

## Widget Reference

| ID | Widget | Panel | Description |
|----|--------|-------|-------------|
| `teacher-welcome-widget` | Welcome | main | Landing page with connection status, quick actions, progress summary |
| `teacher-progress-dashboard` | Progress Dashboard | main | Skill mastery visualization, lesson completion, time analytics |
| `teacher-curriculum-browser` | Curriculum Browser | left | Browse courses, modules, and lessons; start/resume lessons |
| `teacher-canvas-widget` | Canvas | main | Freeform scratchpad for code experiments and notes |
| `teacher-canvas-review` | Canvas Review | main | Review and annotate canvas entries with AI feedback |
| `teacher-learning-analytics` | Learning Analytics | main | Charts and graphs of learning patterns over time |
| `teacher-learning-path` | Learning Path | main | Visual learning path timeline showing progression |
| `teacher-skill-browser` | Skill Browser | left | Browse and search the 306-skill SKILL.md library |
| `teacher-skill-command` | Skill Command | main | Execute skills with input and view results |
| `teacher-ai-history-search` | AI History Search | main | Search through past AI agent conversations |
| `teacher-ghost-timeline` | Ghost Timeline | top | Inline timeline overlay above the editor |
| `teacher-improvement-dashboard` | Improvement Dashboard | main | Track areas needing improvement with actionable items |
| `teacher-permission-mode` | Permission Mode | main | Configure agent permission levels (supervised/auto/plan/autonomous) |
| `teacher-plan-mode` | Plan Mode | main | View and approve agent execution plans before running |
| `teacher-teachable-moments` | Teachable Moments | right | Surface learning opportunities based on current code context |
| `teacher-rewind-panel` | Rewind Panel | main | Browse and restore previous checkpoints |
| `teacher-workflow-builder` | Workflow Builder | main | Create multi-step skill workflows visually |
| `teacher-pulse-panel` | Pulse Panel | bottom | Ambient AI status strip with 2400ms breathing glow (also in teacher-ui) |
