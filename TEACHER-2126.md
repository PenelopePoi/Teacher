# Teacher in 2126

*A reality-anchored projection of the Workshop + Companion IDE, one hundred years from the spec.*

> This document is the north star. The design commitments in the 2026 spec
> were made about the person, not the technology. The 2.4-second breathing
> glow matches resting human respiration. Amber claims attention because of
> the human visual cortex. One indicator carries all state because the human
> attention budget is finite. No spinners because indeterminate waits are
> worse than paced ones. These commitments survive because humans still
> breathe at the same rate.

## What the 2026 designers got right

- The Pulse (2.4s breath, mint idle, amber attention)
- The clip grid (Ableton-native, variant-axis depth)
- The Teachable Moment (gold underline → one-card explainer → personal library)
- Plan Mode as reviewable artifact before execution
- One voice, coalition invisible
- Voice-first intent capture with structured objects
- The spec was about the person, not the hardware

## What the 2026 designers got wrong

- Overestimated remote inference dependency (on-device caught up faster)
- Overestimated Theia's specific architecture persistence (implementation details are archaeological)
- Underestimated how quickly regulation would harden (Plan Mode became law, not best practice)
- Overestimated user appetite for multi-agent visibility (one voice won over eight-agent dashboards)
- Underestimated cooperative/trust model for specialist AI (unions + libraries + labels)

## The commitments that must hold

1. **2.4-second breathing glow** — matches resting human respiration. Never change the cadence.
2. **Amber #E8A948** — the attention color. Across every surface, every century.
3. **No spinners, ever** — indeterminate state = breathing glow.
4. **One indicator carries all state** — the Pulse is the Pulse.
5. **Teacher never pings** — proactive interruption is a flag, not an alert.
6. **The person owns their data** — personal substrate, cryptographic ownership, portable.
7. **Plan Mode before production code** — reviewable artifacts upstream of execution.
8. **Teachable Moments accumulate** — the personal library is the real product.
9. **The tool disappears into the making** — the highest state of an IDE.
10. **The composer in São Paulo is still the customer.**
