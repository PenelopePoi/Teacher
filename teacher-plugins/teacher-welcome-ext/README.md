# Teacher Welcome Walkthrough

A VS Code extension that provides a guided "Getting Started with Teacher" walkthrough for new users.

## Walkthrough Steps

1. **Welcome to Teacher IDE** -- Overview of the IDE and its capabilities
2. **Meet Your Tutor** -- Introduction to the AI tutor sidebar
3. **Your First Lesson** -- Browse and start structured coding lessons
4. **Check Your Work** -- Run assessments with automated feedback
5. **Track Your Progress** -- View learning dashboard with scores and growth

## Commands

| Command                  | Description              |
|--------------------------|--------------------------|
| teacher.openDashboard    | Open Teacher Dashboard   |
| teacher.startTutor       | Start Tutor Session      |
| teacher.browseLessons    | Browse Lessons           |
| teacher.runAssessment    | Run Assessment           |
| teacher.viewProgress     | View Progress            |

## How It Works

The walkthrough appears automatically for new users via the VS Code Getting Started tab. Each step links to a Teacher command that opens the relevant feature.
