# Teacher IDE Theme

Custom dark and light color themes for Teacher IDE, built around the Teacher brand palette of indigo (#6366f1) and purple (#8b5cf6).

## Included Themes

### Teacher Dark
A comfortable dark theme with a warm dark base (#1a1a2e), indigo accents on the activity bar and tabs, and a purple status bar. Optimized for long coding sessions with reduced eye strain.

### Teacher Light
A clean light theme with soft lavender tones, matching indigo/purple accents, and high-contrast syntax highlighting for readability.

## Installation

This extension is bundled with Teacher IDE. The themes are automatically available in the Color Theme picker (Ctrl+K Ctrl+T).
